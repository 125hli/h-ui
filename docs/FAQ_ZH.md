# 常见问题

## 命令行

赋予可执行权限

```bash
chmod +x ./h-ui
```

- 查看系统版本

  ```bash
  ./h-ui version
  ```

- 自定义 Web 端口启动

  ```bash
  ./h-ui server -p [端口]
  ```

# 部署问题

## 项目工程中文件夹的含义

- bin: Hysteria2 的可执行文件和配置文件
- data: 数据库文件
- export: 所有导出的文件
- logs: 系统日志和 Hysteria2 运行日志

## 启动失败

查看日志文件

1. 在命令行查看错误描述
2. 查看 logs/h-ui.log

# 使用问题

## 证书如何管理？

ACME 方式（推荐）系统自动管理。自有证书方式要自己手动维护，到期之后手动替换。

## 自有证书，设置 Hysteria2 证书路径

将证书文件上传至服务器，在配置 Hysteria2 时，填入证书的绝对路径

## 为什么只支持 Hysteria2 版本 >= v2.4.4？

低版本不支持最新 API

## 日志导出失败

没有日志文件
